# 网上购物网站系统 - 概要设计规约

## 1. 项目简介

### 1.1 项目名称
网上购物网站系统

### 1.2 项目概述
本项目是一个基于 B/S 架构的电子商务平台，采用前后端分离的设计模式。前端使用 Vue.js 3 框架构建用户界面，后端使用 Spring Boot 提供 RESTful API 服务，数据库采用 MySQL 8.0。

### 1.3 设计目标
- 构建高可用、高性能的电子商务平台
- 提供良好的用户体验和界面交互
- 实现商品管理、订单处理、数据统计等核心功能
- 支持模块化扩展，便于后续功能迭代

### 1.4 技术栈
| 层次 | 技术选型 |
|------|---------|
| 前端 | Vue.js 3 + Element Plus + Axios |
| 后端 | Spring Boot 2.7 + MyBatis Plus |
| 数据库 | MySQL 8.0 |
| 缓存 | Redis |
| 认证 | JWT |
| 部署 | Docker + Nginx |

## 3. 接口设计

### 3.1 接口规范

本系统采用 RESTful 风格的 API 设计，所有接口返回统一格式：

```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

### 3.2 用户模块接口

| 接口 | 方法 | URL | 描述 |
|------|------|-----|------|
| 用户注册 | POST | `/api/users/register` | 用户注册 |
| 用户登录 | POST | `/api/users/login` | 用户登录 |
| 用户信息 | GET | `/api/users/{id}` | 获取用户信息 |
| 更新用户 | PUT | `/api/users/{id}` | 更新用户信息 |
| 修改密码 | PUT | `/api/users/{id}/password` | 修改密码 |

### 3.3 商品模块接口

| 接口 | 方法 | URL | 描述 |
|------|------|-----|------|
| 商品列表 | GET | `/api/products` | 获取商品列表 |
| 商品详情 | GET | `/api/products/{id}` | 获取商品详情 |
| 创建商品 | POST | `/api/products` | 创建商品（管理员） |
| 更新商品 | PUT | `/api/products/{id}` | 更新商品（管理员） |
| 删除商品 | DELETE | `/api/products/{id}` | 删除商品（管理员） |
| 商品搜索 | GET | `/api/products/search` | 搜索商品 |

### 3.4 购物车模块接口

| 接口 | 方法 | URL | 描述 |
|------|------|-----|------|
| 购物车列表 | GET | `/api/cart` | 获取购物车列表 |
| 添加商品 | POST | `/api/cart/items` | 添加到购物车 |
| 更新数量 | PUT | `/api/cart/items/{id}` | 更新购物车商品数量 |
| 删除商品 | DELETE | `/api/cart/items/{id}` | 删除购物车商品 |
| 清空购物车 | DELETE | `/api/cart` | 清空购物车 |

### 3.5 订单模块接口

| 接口 | 方法 | URL | 描述 |
|------|------|-----|------|
| 订单列表 | GET | `/api/orders` | 获取订单列表 |
| 订单详情 | GET | `/api/orders/{id}` | 获取订单详情 |
| 创建订单 | POST | `/api/orders` | 创建订单 |
| 取消订单 | PUT | `/api/orders/{id}/cancel` | 取消订单 |
| 确认收货 | PUT | `/api/orders/{id}/confirm` | 确认收货 |
| 订单支付 | POST | `/api/orders/{id}/pay` | 订单支付 |

### 3.6 统计模块接口

| 接口 | 方法 | URL | 描述 |
|------|------|-----|------|
| 销售统计 | GET | `/api/stats/sales` | 获取销售统计数据 |
| 商品统计 | GET | `/api/stats/products` | 获取商品统计数据 |
| 用户统计 | GET | `/api/stats/users` | 获取用户统计数据 |
